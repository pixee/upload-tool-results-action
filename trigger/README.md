## Trigger Pixee Action

This action triggers an analysis of pull requests on GitHub using Pixee.

### Inputs

- `url`:
  - **Description:** Endpoint URL where the analysis will be triggered. (Optional)
  - **Required:** No

### Outputs

- `status`:
  - **Description:** The status of the trigger.
